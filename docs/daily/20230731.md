# 2023-07-31__多主题切换方案

## 方案一、link标签动态引入
>提前准备好几套css主题样式文件，在需要的时候，通过动态创建link标签到head中，或者是改变href的属性
### 优点
- 实现了按需加载，提高首屏加载时的速度
### 缺点
- 动态加载样式文件，如果文件过大&&网络不佳时会加载延迟，导致样式切换不刘畅
- 各个主题样式是分开写死的，后续针对某一主题样式更改或者新增很麻烦
